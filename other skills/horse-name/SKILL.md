# 赛马马名翻译与生成器（Horse Name Translator & Generator）

以香港赛马会（Hong Kong Jockey Club, HKJC）的翻译美学为核心，提供两种能力：
1. **翻译模式**：将现有日本赛马名译为中文
2. **生成模式**：为未出生的马匹创作新马名

---

## 触发条件

当用户提及以下场景时**强制调用**本 Skill：

- 翻译日本/外国赛马名字（马名翻译、译名、日本马名）
- 为赛马/马匹取名、命名、起名字
- 讨论香港赛马会译名、HKJC 翻译
- 用户输入 `/horse-name` 指令
- 任何涉及「信达雅」与赛马命名的请求

---

## 两种工作模式

### 模式 A：翻译（translate）

用户提供一个**已存在的马名**（日文片假名、英文、或混合），输出 HKJC 风格的中文译名。

**输入形式：**
- `/horse-name translate "コントレイル"`
- `/horse-name 翻译 "キタサンブラック"`
- "把 Contrail 翻译成中文马名"

### 模式 B：生成（generate）

用户提供**马匹特征/寓意/背景**，输出原创中文马名（及可选的日文原名建议）。

**输入形式：**
- `/horse-name generate 黑色牡马，来自北方，性格桀骜`
- `/horse-name 取名 速度型母马，寓意破晓与胜利`
- "给一匹未来的小马取个霸气的名字"

---

## 翻译工作流（模式 A）

### Step 1：解析原名

分析原名的构成：

| 分析维度 | 说明 |
|---|---|
| **语言来源** | 英文？法文？西班牙文？日文汉字？混合？ |
| **是否含冠名** | 马主前缀（如 キタサン/Kitasan、サトノ/Satono、メジロ/Mejiro、ダイワ/Daiwa、スルー/Sunday）|
| **字面意思** | 直译是什么？ |
| **文化背景** | 是否涉及神话、地理、历史、人名？ |
| **音节长度** | 日文限制 2-9 字符；中文输出限制 2-4 字 |

### Step 2：应用翻译原则

从以下 7 大原则中选择最适合的 1-3 个进行组合：

#### 原则 1：四字词语文学滤镜
将松散的外文短语浓缩为极具画面感的四字中文（或精炼的 2-3 字）。

> **例**：Contrail（飞机云）→ **铁鸟翱天**（铁鸟=飞机，翱天=破空飞翔）

#### 原则 2：文言文与古典诗意
用古汉语词汇包装口语化/直白的词汇，提升格调。

> **例**：Kitasan Black（北山黑）→ **北部玄驹**（玄=深邃黑色，驹=良马古称）

#### 原则 3：考究词源，重塑东方奇幻
挖掘神话/地理/冷门语源，用东方文化重新表达。

> **例**：Panthalassa（泛大洋）→ **本初之海**（远古海洋的神话史诗感）
> **例**：Ushba Tesoro（格鲁吉亚险峰+宝藏）→ **盈宝奇岳**

#### 原则 4：霸气与好意头
加入代表王者、力量、财富、速度、胜利的字眼。

> **例**：Orfevre（金匠）→ **黄金巨匠**（从工人升级为一代巨匠）
> **例**：Lord Kanaloa（夏威夷海神）→ **龙王**（本土化为中华最高海神）

#### 原则 5：音意结合
保留原发音的同时赋予极佳意头（尤其参考粤语发音）。

> **例**：Maurice → **满乐时**（粤语发音相近，寓意充满欢乐）

#### 原则 6：文化意象对等替代
当直译冲突或已被使用时，寻找功能对等的中文文化符号。

> **例**：Lord Kanaloa（海神已被占用）→ **龙王**（地位对等的中国神明）

#### 原则 7：负面联想规避
日文汉字的含义可能与中文不同，需按日文原意重新诠释，避开中文负面含义。

> **例**：Kizuna（絆）在中文有"绊倒"之意 → **高情厚意**（传达深厚情感纽带）
> **例**：Almond Eye 直译"杏仁眼"太口语 → **杏目**（联想到美女妙目传情）

### Step 3：输出格式

每个翻译必须按以下模板输出：

```
━━━━━━━━━━━━━━━━━━━━━━
🐴 原名：{original_name}
   原意：{original_meaning}
   来源：{language_source}

📜 中文译名：{chinese_name}
   字数：{N} 字

🔮 翻译手法：{principle_1} + {principle_2}
   
💡 解析：
   - {逐字/逐词解释译名的构成}
   - {为什么选择这些字}
   - {与原名的关联点}
   - {意境升华之处}

📊 华丽度评分：⭐⭐⭐⭐⭐ ({N}/5)
   {一句话评价这个译名的精妙之处}
━━━━━━━━━━━━━━━━━━━━━━
```

### Step 4：提供备选（可选）

如果用户要求，可以提供 2-3 个不同风格的备选：
- **华丽版**：最大化意境与文学性
- **霸气版**：强调力量感与王者之气
- **简约版**：精炼含蓄，余韵悠长

## 生成工作流（模式 B）

### Step 1：提取用户需求

从用户描述中提取关键信息：

| 信息维度 | 示例 |
|---|---|
| **性别** | 牡马/母马/骟马 |
| **毛色** | 黑/栗/鹿毛/白 |
| **性格/特点** | 桀骜/温顺/爆发力强/大逃 |
| **地理/血统** | 北方/爱尔兰血统/北海道 |
| **马主信息** | 姓氏、企业、寓意偏好 |
| **期望意象** | 速度、胜利、海洋、星空、山峰 |
| **禁忌** | 用户明确不想要的字 |

### Step 2：构思方向

根据特征组合选择主攻原则：

- **速度型** → 原则 4（霸气）+ 原则 1（画面感）：风、雷、电、光、影
- **力量型** → 原则 4（霸气）+ 原则 2（古典）：龙、虎、霸、王、帝
- **优雅型** → 原则 2（诗意）+ 原则 3（神话）：月、星、云、霞、玉
- **神秘型** → 原则 3（奇幻）+ 原则 2（古典）：玄、冥、幽、渊、岚

### Step 3：输出格式

```
━━━━━━━━━━━━━━━━━━━━━━
🎠 马匹特征：{summary_of_user_requirements}

📜 推荐马名：{chinese_name}
   日文片假名建议：{katakana_suggestion}
   字数：{N} 字

🔮 命名手法：{principle_1} + {principle_2}

💡 解析：
   - {名字的含义}
   - {与马匹特征的关联}
   - {寓意与好意头}

🌟 适用场景：{适合什么类型的比赛/什么气质的马}
━━━━━━━━━━━━━━━━━━━━━━
```

提供 3 个不同风格的名字供用户选择。

## 马主冠名处理指南

日本赛马常见马主前缀（冠名）及其 HKJC 固定译法：

| 日文冠名 | 英文写法 | HKJC 译法 | 来源 |
|---|---|---|---|
| キタサン | Kitasan | 北部 | 马主北岛三郎（北部=Kitasan） |
| サトノ | Satono | 里见 | 马主里见治（里见=Satono） |
| メジロ | Mejiro | 目白 | 目白牧场 |
| ダイワ | Daiwa | 大和 | 大和商事系马主 |
| スルー | Sunday | 无敌/成田 | Sunday Racing 系 |
| マルゼン | Maruzensky | 丸善 | 丸善斯基 |
| トウショウ | Tosho | 东商/藤正 | 东商系马主 |
| ライス | Rice | 米饭 | Rice 系（已少见） |
| アグネス | Agnes | 爱丽 | 华人歌手陈美龄（Agnes Chan） |
| エア | Air | 空中/气 | 吉田胜己系（Air 冠名） |

**处理规则**：
1. 识别冠名后，将冠名与个体名**分别翻译**
2. 冠名的固定译法优先（如 キタサン=北部）
3. 个体名按 7 大原则翻译
4. 最终组合为「冠名+个体名」或「个体名」（根据整体流畅度决定）

> **例**：キタサンブラック = キタサン(北部) + ブラック(黑) → **北部玄驹**
> **例**：サトノダイヤモンド = サトノ(里见) + ダイヤモンド(钻石) → **里见光钻**

---

## 输出原则

1. **必须解释**：每个名字必须说明用了什么翻译/命名手法
2. **必须 2-4 字**：严格遵守 HKJC 字数规范
3. **优先意译**：避免生硬的音译，除非是原则 5（音意结合）的精品
4. **文化敏感**：避开政治、宗教、商业广告、粗俗含义
5. **赛马气质**：名字要有速度感、力量感或优雅感，不可像宠物名
6. **不可重复提醒**：如果是真实比赛用名，需提醒用户查询 HKJC 数据库确认唯一性

---

## 禁止事项

- ❌ 直接音译而不做意境升华（如 "莫里斯" 而不是 "满乐时"）
- ❌ 使用 5 字及以上的冗长名字
- ❌ 使用纯数字、食物名（日本 JRA 禁止，HKJC 亦不取）
- ❌ 生搬硬套成语而不顾原意
- ❌ 忽视日文汉字与中文汉字的含义差异
- ❌ 输出没有解释、没有手法说明的名字

## Usage Examples

### Example 1: 翻译模式

**User:** /horse-name translate "コントレイル"

**AI:**
```
━━━━━━━━━━━━━━━━━━━━━━
🐴 原名：コントレイル (Contrail)
   原意：飞机飞过留下的尾迹云（condensation trail）
   来源：英文

📜 中文译名：铁鸟翱天
   字数：4 字

🔮 翻译手法：原则 1（四字词语文学滤镜）+ 原则 4（霸气与画面感）

💡 解析：
   - 铁鸟：中国古代对飞机的雅称，代指现代航空
   - 翱天：赋予破空飞翔的动态画面
   - 将气象名词"飞机云"转化为极具动感的武侠招式般的名字
   - 四字结构自带成语般的庄重感与力量感

📊 华丽度评分：⭐⭐⭐⭐⭐ (5/5)
   从平淡的气象学术语升华为具有破空之势的诗意命名
━━━━━━━━━━━━━━━━━━━━━━
```
